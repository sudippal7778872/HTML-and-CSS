Relative unit percentile(%) how percentile work?
what ever the default value set for that element by browser. that is 100%.
example: here for paragraph tag browser set default size 16px. here for paragraph 100% =16px.

it takes the value from it's parent element value. example: if we write a paragraph under div tag and set it's font size 10px, and then if we give the paragraph tag's font size 100%. then the font size will be 10px.then 100% become 10px got from parent element. not 100%=16px.

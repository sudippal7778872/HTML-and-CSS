Block-level Elements
A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.
A block-level element always takes up the full width available (stretches out to the left and right as far as it can).
we can set height width property in block level elements.
we can set margin for all four sides in block level elements.
we can set padding for all four sides in block level elements.
--------------------------------------------------------------------------
An inline element does not start on a new line.
An inline element only takes up as much width as necessary.
we cant set height and width in inline elements.
we can set margin only two side left and right. top and bottom is not available for inline.
we can set padding in all four side but it is overflow or mixed with other elements in inline.
-----------------------------------------------------------------------------------------------------
now if we want to change any block level element to inline or any inline to block we use display property. ex: display: block. after converting inline element to block element we can apply margin padding etc property.
